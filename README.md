## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/annalatyeva/anna-katsur-portfolio
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```js
   npm start
   ```
